# HRM System

This repository is scaffolded as a two-app workspace:

- `frontend/` - Next.js + Tailwind CSS UI
- `backend/` - FastAPI + PostgreSQL API

## What is included

- Folder structure for login, register, dashboard, departments, positions, employees, and payroll screens
- Reusable UI component slots
- FastAPI app structure with auth, master data, onboarding, and payroll routes
- PostgreSQL-ready model and schema layout
- Environment example files for both apps
